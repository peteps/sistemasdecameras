# 🏭 Câmera 1 — Rastreabilidade Industrial

Sistema de visão computacional para identificação de código de peças.
Pipeline: Frame → YOLO detecta peça → OCR lê código → ERP recebe apontamento.

## Executar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```
